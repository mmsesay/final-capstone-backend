## Getting Started

To get a local copy up and running follow these simple example steps.

## Instructions

```
$ cd <folder>
```

```
$ git clone git@github.com:mmsesay/final-capstone-backend.git
```

```
$ cd final-capstone-backend
```

## Install

```
$ bundle install
```

## Run

```
$ rails db:create
$ rails db:migrate
```

### Usage

Start server with:

```
$  rails server
```

Open `http://localhost:3000/` in your browser.
